# WISP Simulator (Unity Game)

## Project Information

A capstone project developed in partial fulfillment of the requirements for the degree of Bachelor of Science in Computer Science at the National College of Science and Technology.

---

## Overview
WISP Simulator is a 3D first-person game-based training and assessment system designed for Wireless Internet Service Providers (WISPs).

It provides a virtual environment where users can learn, practice, and evaluate real-world WISP installation and maintenance procedures without requiring physical equipment.

---

# Purpose
This project addresses the lack of structured training for aspiring WISP technicians by offering an interactive and scalable simulation platform.

Users can:
- Practice installation procedures safely
- Improve technical skills
- Be evaluated through in-game assessments

---

## Features

### Training Module
- Step-by-step guided simulation
- Covers core WISP procedures

### Assessment Module
- No-guidance mode
- Users perform full installation independently
- Performance is scored

### Quiz Module
- Knowledge-based tests
- Covers theoretical concepts and best practices

### Maintenance Module
- Troubleshooting scenarios
- Simulates common network issues

### Login System
- User authentication required before accessing the simulator

### Web Portal Integration
- Tracks user performance and scores
- Admin dashboard for managing users and data

---

## Simulated Tasks
The simulator focuses on the following WISP operations:

- Antenna Installation  
- Router Setup  
- Cable Crimping  
- Cable Laying  
- Signal Optimization  

---

## Gameplay Details
- **Perspective:** First-person (3D)
- **Mode:** Single-player
- **Flow:** Quiz → Training → Assessment
- **Interaction:** Task-based simulation

---

## System Architecture
- Developed using **Unity Game Engine**
- Desktop-based application
- Connected to a web-based system and database for:
  - User accounts
  - Score tracking
  - Performance monitoring

---

## Benefits
- Reduces training cost for WISPs
- Provides standardized assessment
- Improves technician readiness
- Bridges theory and practical application

---

## Limitations
- No multiplayer support
- Uses predefined scenarios only
- Does not simulate real network traffic or hardware behavior
- Focuses on procedural accuracy
